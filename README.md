## Dossier à récuperer pour tester (ENJMIN)

## INPUT
Forward : Z (or W)  
Move Left : Q (or A)  
Backwards : S  
Move Right : D  
Jump : Space  
Run : Shift  
Crouch : CTRL  
Slide : CTRL (When Running)  
  
## GAMEPLAY  
Wall Run : jump in profile on the wall
Climbing : jump from the front onto the wall
